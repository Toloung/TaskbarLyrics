# Changelog
## Unreleased
### Added
- 任务栏歌词增加专辑封面取色开关

### Changed
- 使用裁剪后的新版立体图标
- 托盘图标主体颜色跟随当前监听的音乐软件变化

## [1.0.9] - 2026-06-24
### Added
- 安装包始终显示安装位置选择页面，可在安装或升级前修改安装目录

## [1.0.8] - 2026-06-23
### Changed
- 软件名称改为 `LyricsBar`
- 整体强调色统一为 `#6ca5fe`
- 托盘右键菜单增加悬浮窗模式快速切换

## [1.0.7] - 2026-06-22
### Added
- 更换应用、托盘和安装包图标
- 增加悬浮窗四角快速放置按钮
- 托盘图标按当前监听播放器显示颜色标记

### Changed
- 悬浮窗设置独立成单独栏目，位置和大小不再受任务栏布局、水平锚点、X/Y 偏移影响
- 悬浮窗支持在窗口内部任意位置拖动，按住 Ctrl 拖右下角可调整大小
- 悬浮窗内歌曲封面和歌词文本会随窗口大小自动缩放
- 封面取色从背景取色改为歌词文字颜色取色

## [1.0.6] - 2026-06-22
### Added
- 增加开机自启动设置开关
- 增加悬浮窗背景按歌曲封面取色开关
- 悬浮窗支持拖动位置和调整窗口大小，并保存位置与尺寸

### Changed
- 强化设置窗口浅色模式的强调色、导航选中态和卡片层次

## [1.0.5] - 2026-06-22
### Added
- 增加设置界面浅色模式
- 增加歌词悬浮窗模式
- 安装器升级时提示将升级现有版本
- 安装前自动检测并关闭正在运行的 `TaskbarLyrics.exe`
- 安装器默认使用简体中文界面

## [1.0.4] - 2026-06-22
### Changed
- 统一发布主程序名称为 `TaskbarLyrics.exe`
- 增加安装包构建脚本与 GitHub Actions 发布流程
- 优化安装包体积并清理旧发布目录，避免旧文件混入

## [0.1.0] - 2026-02-21
### Added
- 初始开源版本
- 任务栏歌词双行滚动
- QQ音乐/网易云/Spotify SMTC 识别
- 多歌词源检索链路
- 设置页字体与外观配置
