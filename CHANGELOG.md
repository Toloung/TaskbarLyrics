# Changelog
## [1.0.6] - 2026-06-22
### Added
- 增加开机自启动设置开关
- 增加悬浮窗背景按歌曲封面取色开关
- 悬浮窗支持拖动位置和调整窗口大小，并保存位置与尺寸

### Changed
- 强化设置窗口浅色模式的强调色、导航选中态和卡片层次

## [1.0.5] - 2026-06-22
### Added
- 增加设置界面浅色模式
- 增加歌词悬浮窗模式
- 安装器升级时提示将升级现有版本
- 安装前自动检测并关闭正在运行的 `TaskbarLyrics.exe`
- 安装器默认使用简体中文界面

## [1.0.4] - 2026-06-22
### Changed
- 统一发布主程序名称为 `TaskbarLyrics.exe`
- 增加安装包构建脚本与 GitHub Actions 发布流程
- 优化安装包体积并清理旧发布目录，避免旧文件混入

## [0.1.0] - 2026-02-21
### Added
- 初始开源版本
- 任务栏歌词双行滚动
- QQ音乐/网易云/Spotify SMTC 识别
- 多歌词源检索链路
- 设置页字体与外观配置
